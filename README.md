## 작업 설명

- jupyter notebook을 통해 데이터프레임 가공 및 데이터 분석을 진행하였습니다
- putty를 통해 AWS EC2 인스턴스에 연결하여 서버를 배포 및 관리하였습니다.
- github action을 활용하여 CI/CD를 구현하였습니다.


## 데이터 설명
	
- Genres = 장르  
- Tags= 태그 
- Content Rating=시청등급  
- Duration = 러닝타임 
- scored by = 리뷰참여자수  
- score = 평점  
- Watchers = 사이트 조회수 
- Popularity = 인기도  
- Episodes = 에피소드 숫자  
- actors = 배우이름  
- platforms = 현재 드라마를 볼 수있는 플랫폼
